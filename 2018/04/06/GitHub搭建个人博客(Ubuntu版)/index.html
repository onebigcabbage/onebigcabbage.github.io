注：这里默认你的电脑可以用Git连接GitHub。如果没有，请点击 git连接github

  • 注册并登录GitHub
  • 新建Repositories
  • 看图，Repositories name和Owner一样，是 ×××××.github.io
  • 然后 ，和我左手右手一个慢动作
  • 点击Choose a Theme
  • 选一个主题，然后Select the Theme。

大功告成，现在在浏览器地址栏输入：

××××××.github.io
一个简(难)约(看)的个人博客就完成了！
